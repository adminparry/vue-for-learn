# vue-element

### 示例使用vue-cli创建project
``` bash
vue create element-project
cd element-project
vue add element
vue add vue-router
vue add vuex
vue add apollo

<!-- 创建router和style文件夹存放路由和样式 -->
cd src && mkdir routers styles 
<!-- 使用sass -->
npm install -D sass-loader node-sass
<!-- 创建store和filter文件夹存放数据和过滤器 -->
mkdir stores filters

```
